# Cloud-Project

Mini 2D farming game with a WinForms launcher + cloud save/logs.

## Download & Install (Windows)
### Option A Setup (Recommended)
1. Download: `setup.exe`
2. Run it and follow the installer steps.
3. After install, launch the app from the Start Menu shortcut.

### Option B ClickOnce
1. Download: `Cloud FinalProject.application`
2. Double-click it to install/run.

> If Windows shows a warning (Unknown publisher), click **More info → Run anyway**.

---

## Requirements
- Windows 10/11
- Internet connection (for cloud features)

---

## What’s inside this repo
This repository contains the **published installation files** (not the full source code):
- `setup.exe` — installer
- `Cloud FinalProject.application` + `Application Files/` — ClickOnce publish output

---

## Troubleshooting
### “Game Not Found”
Make sure the `farming-game` folder is included next to the installed app files (it should be installed automatically).
If you still get this error, reinstall using `setup.exe`.

### Installation blocked / SmartScreen
Windows may warn about apps from unknown publishers:
- Click **More info**
- Click **Run anyway**
